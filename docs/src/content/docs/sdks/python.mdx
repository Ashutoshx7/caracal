`caracalai-sdk` is the main Python package for async agent lifecycle, delegation, Gateway routing, and ASGI context propagation.

## Install

```bash
pip install caracalai-sdk
```

The package requires Python `>=3.14`.

## Configuration entry points

| API | Use it when |
| --- | --- |
| `Caracal.connect()` | Auto-detect a generated profile or environment variables. |
| `Caracal.from_env()` | Load only from `CARACAL_*` environment variables. |
| `Caracal.from_config(path)` | Load a Console-generated `caracal.toml`. |
| `Caracal.from_client_secret(...)` | Build a client that refreshes application subject tokens through STS. |

```python
from caracalai_sdk import Caracal

caracal = Caracal.connect()
```

## Core methods

| Method | Purpose |
| --- | --- |
| `async with caracal.spawn(...)` | Create and bind an agent session. |
| `async with caracal.delegate(...)` | Create and bind a delegation edge to an existing agent session. |
| `async with caracal.delegate_to_spawn(...)` | Spawn a child session and create the delegation edge before yielding context. |
| `caracal.headers(allow_root=False)` | Project the current context into HTTP headers. |
| `async with caracal.bind(ctx)` | Rebind a captured context into a new async task. |
| `async with caracal.bind_from_headers(headers, allow_root=False)` | Bind inbound Caracal envelope headers. |
| `caracal.transport(allow_root=False, **kwargs)` | Return an `httpx.AsyncClient` that injects Caracal headers and Gateway routing. |
| `caracal.sync_transport(allow_root=False, **kwargs)` | Synchronous `httpx.Client` counterpart. |
| `caracal.context_middleware()` | ASGI context propagation middleware factory. |
| `caracal.gateway_request(resource_id, path="/")` | Build a Gateway URL and `X-Caracal-Resource` header. |

## Delegation constraints

```python
from caracalai_sdk import DelegationConstraints

constraints = DelegationConstraints(
    max_hops=1,
    budget=5,
    policy_approved=True,
)
```

`DelegationConstraints` uses Python field names: `resources`, `max_depth`, `max_hops`, `ttl_seconds`, `budget`, `policy_approved`, `expires_at`, and `broad_reason`.

## ASGI propagation boundary

`context_middleware()` binds a Caracal envelope after a verifier boundary. It does not verify JWT signatures, audience, scopes, token use, or revocation. Pair it with a Gateway, connector, or MCP transport verifier before trusting the context.
